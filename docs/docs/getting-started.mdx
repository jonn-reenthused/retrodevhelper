## Prerequisites

Before installing retrodevhelper you will need the following applications installed

- [Homebrew](https://brew.sh) - Package Manager for macOS
- [Docker Desktop](https://www.docker.com/products/docker-desktop/) - Container Platform for macOS

## Installation

With everything installed you can run the following commands below to install retrodevhelper.

```bash
brew tap studioripe/retrodevhelper
brew install retrodevhelper
```
